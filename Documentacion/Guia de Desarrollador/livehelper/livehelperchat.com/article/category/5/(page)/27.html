  • 2014-04-12 21:30:31

    New version - 1.95v new module!

    Mega release.

    Sponsored features

    1. Browse offers

    Other features

    1. Hidden option for department, usefull if you want to have fancy department workflow and let say some department to which chat is transfered you wish were invisible to user.
    2. Operator won't see others departments visitors in online visitors. Also updated javascript code generation to have this. Also operators will be able to edit only their own departments. Very usefull if you use each department for different client.
    3. Option to close embeded chat if operator has closed it faster then user closed it.
    4. Browser notification about new chat will show client nick and last three messages instead of the same message all the time.

    In general these changes
