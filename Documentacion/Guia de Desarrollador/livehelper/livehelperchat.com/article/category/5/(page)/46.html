  • 2013-07-25 19:22:58

    New version 1.41v pro active invitation messages by identifier and others changes

    Features:

    1. Site identifier in code generation. It can be used to have different invite messages for different domains etc.
      1. https://github.com/remdex/livehelperchat/issues/73
    2. Option to turn of online users tracking but completely remove pro active chats invitations table scan and save a little bit of performance.

    Enhancements:

    1. Template cache by port.
      1. https://github.com/remdex/livehelperchat/issues/66
    2. Simpler permission access generation.
      1. https://github.com/remdex/livehelperchat/issues/62

    Bugfixes:
    1. User who did not had permission to close chat was able to do that because of typo error. It's fixed now.

    execute doc/update_db/update_28.sql and follow standard upgrade instructions

    Additional notices

    If you executed database update please check that you have enabled pro active chat messages invitations like from now it's configurable option. Also read updated documentation to find out how to use identifier

  • 2013-07-20 21:48:00

    If anyone wonders why I'm doing this

    Just thought it would be interesting to share the reason why I'm spending enourmous amount of time on this project.

    1. It's fun to see increasing number of installs and how chat is being customized.
    2. At the time I started this project there did not existed really quality Open Source product for Live Support software. Do you know any open source product with sutch code quality and with so many features? At the moment I know only two real Open Source alternatives
      1. http://mibew.org/ - has not been updated for over two years i ques, correct me if I'm wrong
      2. http://www.craftysyntax.com/ - design like from 2000 years :D
    3. Perhaps in the future I will think how to make little money from all this, but you can be sure it will never be closed code project and will always remain Open Source. Perhaps some day I will add automated hosting etc.

    Let me know in comments what you think in general about Live Helper Chat :)
